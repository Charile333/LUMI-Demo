## 📊 状态变化

### 加载状态
```
YES 价格加载中:
┌──────────────┐
│     YES      │
│              │
│     ...      │  ← 脉冲动画 (animate-pulse)
└──────────────┘

交易按钮提交中:
┌──────────────────────┐
│  ⏳ 处理中...        │  ← 旋转动画 (animate-spin)
└──────────────────────┘
```

### 禁用状态
```
输入框禁用:
┌──────────────────┐
│  $ [disabled]    │  ← 透明度 50% + 禁止光标
└──────────────────┘

按钮禁用:
┌──────────────────┐
│  [  disabled  ]  │  ← 透明度 50% + 无悬停效果
└──────────────────┘
```

## 🎭 动画效果

### 弹窗打开动画
```
步骤 1: 背景淡入
  opacity: 0 → 1 (200ms)

步骤 2: 卡片缩放 + 淡入
  scale: 0.95 → 1 (300ms)
  opacity: 0 → 1 (300ms)

效果: animate-in fade-in zoom-in-95
```

### 按钮悬停动画
```
正常状态:
  scale: 1
  shadow: base

悬停状态:
  scale: 1.02
  shadow: enhanced

过渡: transition-all (200ms)
```

## 🌈 渐变效果

### 按钮渐变
```css
/* Buy 按钮 (选中) */
background: linear-gradient(to right, #10b981, #059669)
/* emerald-500 → emerald-600 */

/* Sell 按钮 (选中) */
background: linear-gradient(to right, #f43f5e, #e11d48)
/* rose-500 → rose-600 */

/* YES 选中 */
background: linear-gradient(to bottom right, #10b981, #059669)
/* emerald-500 → emerald-600 */

/* NO 选中 */
background: linear-gradient(to bottom right, #f43f5e, #e11d48)
/* rose-500 → rose-600 */
```

### 阴影效果
```css
/* YES 选中阴影 */
box-shadow: 0 10px 15px -3px rgba(16, 185, 129, 0.3)

/* NO 选中阴影 */
box-shadow: 0 10px 15px -3px rgba(244, 63, 94, 0.3)

/* 按钮悬停阴影 */
box-shadow: 0 20px 25px -5px rgba(current-color, 0.3)
```

## 📱 响应式设计

### 平板端 (640px - 1024px)

## 🖼️ 实际使用场景

### 场景 1: 快速买入 YES
```
用户操作流程:
1. 点击市场卡片的"交易"按钮
2. 弹窗打开，默认 Buy + YES
3. 输入金额（或使用快速按钮）
4. 查看预估收益
5. 点击"买入 YES"按钮
6. 完成交易
```

### 场景 2: 卖出 NO
```
用户操作流程:
1. 点击市场卡片的"交易"按钮
2. 点击"Sell"按钮
3. 点击"NO"卡片
4. 输入卖出数量
5. 查看预估信息
6. 点击"卖出 NO"按钮
7. 完成交易
```

## ✨ 细节优化

1. **对勾图标**: 选中状态右上角显示 ✓
2. **金额前缀**: 输入框左侧固定显示 $
3. **脉冲动画**: 加载时使用脉冲效果
4. **旋转动画**: 提交时显示旋转加载
5. **缩放效果**: 悬停时按钮轻微放大
6. **阴影增强**: 悬停时阴影加深
7. **颜色过渡**: 所有颜色变化使用平滑过渡
8. **禁用提示**: 禁用状态降低透明度

---

**提示**: 以上设计遵循现代 UI/UX 最佳实践，确保良好的用户体验和视觉效果。
